Installing the reader extension

31 Oct. 2015

Getting started with reader is pretty simple. Run the following cells of code in any notebook to download and install the extension. The output is also shown below.

In [1]:
import notebook.nbextensions
reader_js = 'https://gist.githubusercontent.com/jameslmartin/b52f4778782fa4a61dbd/raw/c749190cc3471207af6726ab10b9dd6ba4bf239f/reader.js'
notebook.nbextensions.install_nbextension(reader_js, user=True)

Keyboard shortcuts

Click Help > Keyboard Shortcuts to see your available list of shortcuts. Currently, we support three functions:

  • Shift-M to read if the mode is currently 'Command' (useful for navigation)
  • Shift-R to read the highlighted cell (currently only possible in Command mode)
  • Shift-ESC to cancel reading of the cell (useful for code with links)
