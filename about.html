What is this site?

This is the landing page for the Jupyter a11y project.

Currently, I am working to:

  • make screen reading more efficient using Google's WebSpeech API and keyboard shortcuts
  • read markdown, code, and raw convert cells differently
  • read code cells intelligently based on syntax highlighting
  • play earcons to reflect kernel state
  • expand interaction with notebooks through dictation
  • include new CSS styling to help visually impaired users
